# AI Usage Declaration

This document outlines how AI tools were utilized during the development of the Gemstone Recommendation App.

## Tools Used
- **Antigravity (AI Coding Assistant)**: Acted as the primary pair-programming partner.
- **Underlying AI Model**: Gemini 3.1 Pro (High) 

## How AI Was Used
1. **Scaffolding and Setup**: The AI assisted in quickly scaffolding the Vite + React + TypeScript project and setting up the initial file structure.
2. **Architecture and Planning**: The AI drafted an implementation plan breaking down the problem statement into logical steps: defining the data model, UI components, and state management logic.
3. **Styling and Aesthetics**: The AI generated premium Vanilla CSS using modern design practices (CSS variables, flexbox/grid layouts, micro-animations, and a sleek dark theme) to meet the aesthetic requirements.
4. **Code Generation**: The AI was used to write the React components (`App.tsx`, `Questionnaire.tsx`, `Recommendation.tsx`) and implement the recommendation logic based on user input.

The use of AI allowed for rapid prototyping, ensuring clean code structure, and meeting the delivery deadline while adhering to best practices.
